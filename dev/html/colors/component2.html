Brand Styles

Colors

Colors are based upon the Encore Logo Style Guide. We should design our interfaces in a manner that keeps with our design principles of Visibility and User Focus, by using vivid colors for highlighting important parts of our apps, but keeping the design clean and simple. Also, by applying the principle of Simplicity, we want to keep everything with a flat look, so the colors should be solid, with no gradients.

Primary Blue
HEX code: #0069AA
SASS variable name: $primaryBlue

Usage: Used in headings, header background, buttons, links, active menu items, and other elements.

Light Primary Blue
HEX code: #0079C4
SASS variable name: $lightBlue

Usage: For accentuating the Primary Blue. For example, the header is composed from a Primary Blue section and a Light Blue section.

Secondary Blue
HEX code: #133B62
SASS variable name: $darkBlue

Usage: For button hover states, and for the same as the Primary Blue elements, but in cases in which a greater contrast is required.

Primary Green
HEX code: #7AC143
SASS variable name: $primaryGreen

Usage: For underlines, active breadcrumb page, navigation headings, item accents, and for special emphasis in copy.

Light Green
HEX code: #88c756
SASS variable name: $lightGreen

Usage: For accentuating the Primary Green.

Secondary Green
HEX code: #5E9732
SASS variable name: $darkGreen

Usage: For the same as the Primary Green items, but in cases in which greater contrast is required.

Copy color
HEX code: #58595b
SASS variable name: $textColor

Usage: Primary copy color for white and light backgrounds.

Disabled color
HEX code: #AAAAAA
SASS variable name: $disabledGray

Usage: Disabled and "empty" input fields. Can also apply for legal footers, special copy elements, or other cases in which a lighter and less contrasting font is required.

White
HEX code: #FFFFFF
SASS variable name: $white

Usage: Cards/forms backgrounds, main color for copy inside dark backgrounds.

Black
HEX code: #000000
SASS variable name: $black

Usage: Only for special cases in which the main Copy Color does not provide enough contrast.

Background
HEX code: #F5F5F5
SASS variable name: $bgColor1

Usage: Main site background. Can also be the BG color for some widgets or special elements.

Light Gray
HEX code: #E3E3E3
SASS variable name: $lightGray

Usage: For separators in menus, or some elements' borders in order to provide adequate contrast against light backgrounds (however, a box-shadow is preferred in this case when appropriate)

Red
HEX code: #FF002A
SASS variable name: $red

Usage: For "required" copy, error messages, and special copy highlighting.

Pink
HEX code: #FF99AA
SASS variable name: $pink

Usage: For "required" copy, error messages, and special copy highlighting, over a dark background.
