Brand Styles

Fonts

As per the Logo Style Guide, the corporate fonts for Encore documents is the following:

  • Helvetica Neue Bold (Headers & Subheads)
  • Helvetica Neue Regular (Body Copy)

With fallback to Arial when Helvetica is not available in the system:

  • Arial Bold (Headers)
  • Arial (Body Copy)

CSS rule

body {
  font-family: 'Helvetica Neue', Helvetica, Arial, sans-serif;
}

Text Size

  • We should define the font sizes in device-independent pixels (px in CSS).
  • The default font size for the site should be 16px, for optimal readability in most screen sizes.
  • Secondary texts, like annotations in the footer, or similar, can have a slightly smaller font size of 14px

Font weights, decorations, styles

  • Copy should have a normal font-weight.
  • Highlighted words should be bold
  • Quotes, foreign words, or similar, should be italicized
  • There will be no text underlines. Some headings and special formatted text can have a simulated underline using border-bottom property in CSS
  • Links should be bold and have the Primary Blue color, or White when over dark backgrounds.
  • Error text, or required field indicators, should be red and bold.
