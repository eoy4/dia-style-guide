Layout

General Layout

We should keep our principle of Consistent and Familiar Interface throughout the site. The site's general layout will include the following elements:

A header

  • The header will include the logo, along with any account-related items (sign in, register, my account).
  • In mobile devices, the header will become a title bar, with a Menu button on the left.

A breadcrumb

  • Positioned below the header.
  • The breadcrumb will be replaced with a "home" button in mobile devices, since tapping on breadcrumb links can be difficult for small screens.

A left sidebar

  • Contains main navigation.
  • Current page is highlighted.
  • Main navigation can be replaced with controls for a specific tool.
  • In smaller screens the sidebar disappears and turns into a drawer-style menu that opens/closes.

A main content section

  • All the content will be placed here.
  • Extends from where the left sidebar ends to the end of the viewport. No maximum width is specified.
  • In smaller screens the left sidebar disappears, so the main content section spans the whole width of the viewport.

A footer

  • This will contain copyright info, logo, and pertinent links and info.

The site's layout will utilize a customized version of Twitter's Bootstrap library, specifically the responsive utility classes and the grid system. Please refer to the Bootstrap CSS guide for information on how to use these features.



We should not use any of Bootstrap's default styles or themes. All the styling is custom made and following the guidelines outlined in these design pages.
