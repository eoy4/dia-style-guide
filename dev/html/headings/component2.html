Type & Controls

Headings

Use headings semantically according to context. Do not use heading tags as purely cosmetic, use them in a coherent hierarchy. This ensures that we keep with our principle of Structure, providing both semantical and visual organization to our pages.

This is an H1

  • Font size is 26px
  • Color is Primary Blue
  • Border bottom is (SASS code)
    border-bottom: 2px solid $primaryGreen;
  • It is used for the main page title. Whenever possible, only one H1 should be present in the page.

This is an H2

  • Font size is 22px
  • Color is Primary Blue
  • Use for sub-topics one level below the main page topic.
  • Also use for any unrelated content bits in the page that require a heading (for example in disclaimers, ads, toolbars, etc.
