Iconography

Icons

We will use the Material Design Icons from Google. These icons are free and open source, and have a simple, modern, bold, and appealing design. There are more than 900 icons to choose from, and they can be styled with CSS, and are vector-based, so they look great in any resolution, in any screen.

Google's Material Icons can be downloaded and used in a variety of formats. There are png images, svg images, and a web font. We are implementing the web font, hosted locally as opposed to Google's CDN. To download the web font, go here. Here is an example of the CSS needed for this:

@font-face {
  font-family: 'Material Icons';
  font-style: normal;
  font-weight: 400;
  src: url("fonts/material-icons/MaterialIcons-Regular.eot"); /* For IE6-8 */
  src: local('Material Icons'),
       local('MaterialIcons-Regular'),
       url("fonts/material-icons/MaterialIcons-Regular.woff2") format('woff2'),
       url("fonts/material-icons/MaterialIcons-Regular.woff") format('woff'),
       url("fonts/material-icons/MaterialIcons-Regular.ttf") format('truetype');
}

.material-icons {
  font-family: 'Material Icons';
  font-weight: normal;
  font-style: normal;
  font-size: 24px;  /* Preferred icon size */
  display: inline-block;
  line-height: 1;
  text-transform: none;
  letter-spacing: normal;
  word-wrap: normal;
  white-space: nowrap;
  direction: ltr;

  /* Support for all WebKit browsers. */
  -webkit-font-smoothing: antialiased;
  /* Support for Safari and Chrome. */
  text-rendering: optimizeLegibility;

  /* Support for Firefox. */
  -moz-osx-font-smoothing: grayscale;

  /* Support for IE. */
  font-feature-settings: 'liga';
}

An their HTML usage is like this:

<!-- This will only work on modern browsers (IE10+) -->
<i class="material-icons">credit_card</i>

<!-- This will work on all browsers -->
<i class="material-icons">&#xE870;</i>

To view all available icons and to read about their usage, please refer to this page and this guide.
