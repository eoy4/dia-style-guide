Encore DPP DIA - Digital Resource Guide (DRG)

Introduction

This online Digital Resource Guide aims to be a centralized place in which all the different elements that are being designed for the DIA project will be implemented, in order for developers to have a quick reference and code samples for their usage.

In the Design Principles section, we explain the principles that act as our pillars for the site's design and structure, and in each section we discuss how these principles tie in with the whole.

These pages go alongside the Web Design Document, which will contain the same information, but in a printable, shareable format.
