Tabular Data

Tabular data is used for presenting data that comes either from a file, a database, or other types of documents that need a clear and consistent presentation, for better understanding and processing. Tabular data fits perfectly inside cards, this ensures it keeps delimited and is cleanly presented. However, this type of data can be presented inside any container, not only cards.
